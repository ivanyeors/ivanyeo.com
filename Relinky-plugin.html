Relinky: Streamlining Design System Workflow

Relinky is a Figma plugin that automates the detection and management of unlinked design tokens and variables, significantly reducing design debt and improving team efficiency.

×

As a senior product designer working with large-scale design systems, I came to realise a critical pain point during a handoff process across differeent design teams:

Design System Consistency

Maintaining consistency across large-scale design systems was challenging and time-consuming.

Handoff Process

The handoff process was not seamless and required extensive back-and-forth between designers and developers.

The Design Challenge

Design systems in large organizations often face several challenges:

Design Tokens

  • Inconsistent use of design tokens across files and teams
  • Time-consuming manual audits of design files

Components

  • Design drift between components and their instances
  • Difficulty in maintaining design system compliance

Smart Detection System

×

Automated Token Detection

  • Unlinked color values and strokes
  • Typography inconsistencies
  • Spacing and layout variations
  • Component instance deviations

Variable Management System

×

Comprehensive Variable Control

The plugin provides powerful tools for managing design variables:

  • Real-time library status monitoring
  • Batch selection and updating of variables
  • Visual preview of variable values
  • Direct library activation from results

Impact on Design Workflow

Based on real-world implementation, Relinky has demonstrated significant improvements in design team efficiency:

  • 80% reduction in time spent on design system audits
  • 95% improvement in design system compliance
  • Streamlined collaboration between designers and developers
  • Enhanced quality assurance for design deliverables

Continuous Evolution

The plugin is continuously evolving with planned features including:

  • Advanced variable scanning capabilities
  • Enhanced typography grouping system
  • Comprehensive padding controls
  • Integrated feedback system with JIRA
