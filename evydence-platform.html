Current Evydence Platform Issues

User feedback has identified several specific issues with the current Evydence platform:

×

Navigation

  • Preference for the search tree on the Insight platform over Evydence
  • Navigation improvements needed for features like 'My Collections'

UI/UX

  • Revert 'Insert' button to 'Copy' to prevent unintended code insertion
  • Some users prefer using interfaces and functions on other platforms, such as Yanagishima, over Evydence

Data Manipulation

  • Develop a sandbox feature for clients to manipulate joined data freely, reducing reliance on external tools like Excel
  • Implement left join table functions and allow joining of original and/or unified data
